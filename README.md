easyDownload
============


##说明##

下载包, 目前只支持HTTP下载, 包括断点, 多线程, 下载限速, 下载优先级, 一个地址从多个服务器下载等功能.

使用前调用 DownloadPool.init(context); 进行初始化, 之后将可以开始下载.



##gradle使用说明##

compile 'com.github.lisicnu:easyDownload:0.2.0'


// 内部已经自动引用
